Simple Scientific Calculator

Overview
The Simple Scientific Calculator is an Android application designed to perform basic arithmetic operations and scientific calculations, including trigonometric functions. The app features a user-friendly interface with a sleek design, making it easy for users to input values and view results. It supports both degree and radian modes for trigonometric calculations.

Features
Basic Arithmetic Operations: Addition, subtraction, multiplication, and division.
Scientific Functions: Supports trigonometric functions (sin, cos, tan) and their inverses (arcsin, arccos, arctan).
Reciprocal Trigonometric Functions: Calculates cosecant, secant, and cotangent.
User-Friendly Interface: Intuitive layout with buttons for easy access to functions.
Dynamic Mode Switching: Toggle between degrees and radians for trigonometric calculations.
Input Validation: Alerts for invalid inputs and operations (e.g., division by zero).

Technologies Used
Android SDK: The application is built using Java and the Android SDK.
XML Layouts: User interface designed with XML for structured layout management.
Gradients and Shadows: Enhanced visual appeal with gradient backgrounds and button shadows.

Installation
Clone the repository or download the project files.
Open the project in Android Studio.
Build and run the application on an Android device or emulator.

Usage
Launch the app on your Android device.
Enter values in the input field.
Select the desired operation using the buttons.
For trigonometric functions, ensure to set the mode (degrees or radians) using the switch.
Press the "=" button to calculate the result, which will be displayed in the input field.
